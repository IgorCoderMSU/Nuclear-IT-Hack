Проект по классификации тканей мозга (control / endo / exo) по Рамановским спектрам в двух диапазонах:

- **1500 cm⁻¹** (белки, амиды и др.)
- **2900 cm⁻¹** (липиды, CH-растяжения)

Модель определяет класс каждого пикселя спектрального изображения, а затем выдаёт наиболее вероятный диагноз для всего среза (по большинству голосов пикселей).

## Основные файлы в репозитории

| Файл                            | Что делает                                                                 | Когда использовать                                 |
|-------------------------------|----------------------------------------------------------------------------|----------------------------------------------------|
| `predict.py`                  | Основной скрипт для предсказания на **новых** данных                        | Когда нужно классифицировать новый .txt / .csv файл |
| `Untitled2 (6).ipynb`         | Jupyter-тетрадь: обработка, обучение, валидация, feature importance        | Для воспроизведения обучения и экспериментов       |
| `model_1500.pkl`              | Обученная модель LightGBM для диапазона ~930–2000 cm⁻¹                     | Автоматически используется в `predict.py`          |
| `model_2900.pkl`              | Обученная модель LightGBM для диапазона ~2600–3200 cm⁻¹                    | Автоматически используется в `predict.py`          |
| `cols_1500.pkl`               | Список длин волн (столбцов), на которые интерполируются спектры 1500       | —                                                  |
| `cols_2900.pkl`               | Список длин волн для диапазона 2900                                        | —                                                  |
| `submission_result.csv`       | Пример результата работы `predict.py` (попиксельные предсказания)         | Создаётся автоматически при запуске                |
